### 1. **The Core Blocks: Jamo vs. Syllables**
Korean can be encoded in two main ways:
* **Hangul Jamo (자모):** The individual consonants (초성, 중성) and vowels (종성).
* **Hangul Syllables (완성형):** Pre-composed, whole syllable blocks (e.g., 한, 글).
Unicode supports **both**, but the syllabic blocks are by far the most common and recommended for normal text.
---
### 2. **Hangul Syllables Block (가–힣) - The Most Important**
This is the contiguous block containing **11,172 pre-composed modern Hangul syllables**.
* **Unicode Range:** `U+AC00` to `U+D7A3`
* **Example:** `한` (U+D55C), `글` (U+AE00)
* **How it works:** The block is algorithmically arranged. The first syllable, **가** (U+AC00), is formed by the first consonant (ᄀ), first vowel (ᅡ), and no final consonant. The ordering is systematic:
* Initial consonant (Choseong) changes slowest.
* Medial vowel (Jungseong) changes next.
* Final consonant (Jongseong) changes fastest.
* **This is the standard way** Korean text is stored and displayed in digital systems. Any modern Korean font will support this entire block.
---
### 3. **Hangul Jamo Blocks (Individual Components)**
These are used for rare academic, phonetic, or archaic text representation where you need to show individual letter components separately.
* **Hangul Jamo (U+1100–U+11FF):** The set of **consonants and vowels** used for forming Old Korean or for explicit phonetic display.
* **Examples:** `ᄒ` (U+1112), `ᅡ` (U+1161), `ᆫ` (U+11AB)
* If you typed these in sequence (한), a smart renderer *might* combine them into **한**, but they are stored as three separate code points.
* **Hangul Compatibility Jamo (U+3130–U+318F):** These exist primarily for **round-trip compatibility** with older Korean standards (KS X 1001). **You should not use these for normal modern text.**
* **Examples:** `ㅎ` (U+314E), `ㅏ` (U+314F), `ㄴ` (U+3134)
* They appear as standalone letters, like on a keyboard.
* **Hangul Jamo Extended-A/B (U+A960–U+A97F, U+D7B0–U+D7FF):** Additional jamo for transcribing obsolete Korean sounds, mostly used for academic purposes in writing Middle Korean (e.g., the `ᅀ` series, doubled consonants).

**For virtually all purposes—web, databases, documents—you should use the pre-composed Hangul Syllables block.**
