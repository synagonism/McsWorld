"An **abugida** is a type of writing system in which each symbol typically represents a **consonant with an inherent vowel**, and other vowels are denoted by modifying the base symbol. It sits between alphabets and syllabaries in the spectrum of writing systems.
---
### ğŸ”¤ Key Features of Abugidas:
1. **Consonant-Vowel Structure**:
* Each character generally represents a consonant followed by a default vowel (often "a").
* To indicate a different vowel, the character is modified with diacritics or additional marks.
2. **Inherent Vowel**:
* A base consonant like "k" might be understood as "ka" unless a diacritic changes the vowel (e.g., to "ki" or "ku").
3. **Vowel Suppression**:
* A special mark (often called a *virama*) is used to remove the inherent vowel, indicating a "pure" consonant sound.
4. **Syllabic Nature**:
* While the system is organized by consonants, it is read as syllables.
